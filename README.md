# covid19-notifier

!! Please Read the LICENSE

Send SMS Notification : https://covid19notifier.devs-cast.com

## install
```bash
$ git clone https://github.com/devscast/covid19-notifier.git covid19notifier
$ cd covid19notifier

$ php artisan migrate // set up database in .env
$ php artisan db:seed
$ php artisan serve
```
